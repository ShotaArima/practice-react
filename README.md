# Practice React
このリポジトリでは、React, TypeScriptなどを練習するリポジトリ



## ディレクトリ構造

```
docker-react/
├── app/
│   ├── public/
│   ├── src/
│   ├── package.json
│   ├── package-lock.json
│   ├── .gitignore
│   └── README.md
├── docker/
│   └── Dockerfile
└── compose.yaml
```

- `public/`
    - index.htmlなどの外部から見られるものが表示される
- `src/`
    - コンポーネントなどの情報や関数定義などを行う
